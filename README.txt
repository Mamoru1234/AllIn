Реєстрація:
Для початку сервіст повинен зареєструватись як клієнт(client)
цього застосунку через головну сторінку.
В результаті реєстрації клієнт отримує ід(clientID) та пароль(clientSecret)

Після реєстрації клієнта реєстрація юзерів в сервісі-клієнті відбуваєтся наступним чином:

GET: /oauth/authorize
    client_id - ід сервісу-клієнта
    redirect_url- урл який отримаю грант токен в реквесті
    {
        надалі відбуваєтся визначення особи юзера та підтвердження з боку юзера надання
         прав сервісу-клієнту
    }
GET: redirect_url
    grant_token - тимчасовий токен надання доступу

POST: oauth/token:
    client_id
    client_secret,
    grant_token
    RESPONSE:
        access_token - не тимчасовий

GET: /api/{access_token}/general_info
